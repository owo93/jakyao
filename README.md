# Jakyao

A simple countdown for Suankularb Wittayalai graduation.

## Usage

To use this application, configure your `.env` accordingly,

| key                  | value                                             |
| -------------------- | ------------------------------------------------- |
| `TARGET_JAKYAO_DATE` | Your expected graduation date `YYYY-MM-DDTHH:mmZ` |
| `SK_GENERATION`      | Suankularb generation, as a number                |
